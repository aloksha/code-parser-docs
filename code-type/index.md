# Supported code types

Dynamsoft Code Parser(DCP) supports parsing following code types:

- [North American Driver License](na-dl.md)
- [South African Driver License](sa-dl.md)
- [Aadhaar card](aadhaar.md)
- [MRZ](mrz.md)
- [VIN](vin.md)
